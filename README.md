# js-roadmap
Daily JavaScript practice & projects.

## Struktur
- day1, day2, ...  — latihan harian
- projects/        — project yang lebih besar
- notes/           — catatan, snippet, referensi

## Cara kontribusi
1. Buat branch: `git checkout -b feat/day1`
2. Commit: `git add . && git commit -m "Day 1 - ..."`
3. Push: `git push -u origin feat/day1`
4. Buka PR ke `main`.
